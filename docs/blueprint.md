# **App Name**: Campus Compass

## Core Features:

- Campus Selection: Allow users to select a campus from a list of available campuses. Initially, only CSUS will be fully functional.
- Building Search: Enable users to search for buildings within the selected campus by name, abbreviation, or alias.
- Building Details: Display detailed information about a selected building, including its name, abbreviation, description, and navigation options.
- Navigation Integration: Integrate with native map applications (Apple Maps on iOS, Google Maps on Android) to provide turn-by-turn navigation to selected buildings.
- Data Loading and Caching: Load campus and building data from local JSON assets and cache it for offline access and improved performance.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional, academic feel.
- Background color: Light gray (#F5F5F5), nearly white but easier on the eyes.
- Accent color: Amber (#FFC107) to highlight key interactive elements.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif.
- Use clean, minimalist icons sourced from Material Design Icons for a consistent and modern look.
- Employ an 8pt spacing grid throughout the application to maintain visual consistency and rhythm.
- Implement subtle fade and slide animations for screen transitions and smooth bottom sheet presentations.
